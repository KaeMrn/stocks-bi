# 📁 routes

Contient la définition centralisée des routes de l'application.

Exemples :
- `routes.tsx` → déclaration des routes
- `ProtectedRoute.tsx` → gestion des guards (auth, permissions)
- `paths.ts` → toutes les routes en constantes (`/login`, `/dashboard`, etc.)

🎯 Objectif : centraliser la navigation pour éviter la duplication.
